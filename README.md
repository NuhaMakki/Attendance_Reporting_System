# Attendance Reporting System for Trainees
<p align="center"> <img src="GAIT_KAU_Logo.png" width="20%" alt="General Administration of Information Technology"> </p>

## 🚀 Project Overview
A **desktop application** designed to efficiently manage trainee attendance by retrieving data from a **fingerprint scanner device** and generating detailed reports. Built during my summer training at **General Administration of Information Technology (GAIT), King Abdulaziz University (KAU)**, this system simplifies attendance tracking and report generation.

## 🎯 Key Features
- ✅ **Automated Attendance Tracking:** Processes fingerprint data to manage entry and exit times.
- ✅ **Report Generation:** Export attendance records in **PDF and Excel formats**.
- ✅ **Data Management:** Add, update, or delete trainee and department records.
- ✅ **Error Handling & Validation:** Ensures data accuracy and prevents incorrect entries.

## 🛠️ Tech Stack
- **Programming Language:** C#
- **Framework:** .NET
- **Database:** Microsoft Access

## 🚀 Getting Started  
### Installation & Setup
1️⃣ Clone the repository:  
   ```sh  
   git clone https://github.com/NuhaMakki/Attendance_Reporting_System.git  
   ```  
2️⃣ Open the project in **Visual Studio**.  
3️⃣ Replace the existing `.mdb` file in the `database` folder with your fingerprint scanner database (if needed).  
4️⃣ Run the application.

- **Track Attendance:** Connect the fingerprint scanner and record trainee attendance automatically.
- **Generate Reports:** Select a date range and export reports as **Excel or PDF**.
- **Manage Data:** Add, update, or remove trainee and department records.
